## Course Structure

The course is organised into 4 Learning Units (LU1–LU4), each with in-class activities that build toward the Day 2 Case Study assessment:

| Learning Unit | Focus | Activities |
|---|---|---|
| **LU1 — Stakeholders and Organisation** | Internal/external stakeholders, audience types, brand design, reputation assessment | 1–4 |
| **LU2 — Customer Influence** | Customer perceptions, brand reputation, campaign reception, active listening | 5–8 |
| **LU3 — Branding in Marketing** | Branding basics, role in marketing, awareness campaigns, PR execution | 9–12 |
| **LU4 — Branding Effectiveness** | Reputation metrics, success indicators, PR tactics, KPI monitoring | 13–17 |

📖 **Full activity instructions:** see [`labs/README.md`](labs/README.md) for all 17 in-class activities, and [`labs/tools.md`](labs/tools.md) for the frameworks used. Slides, the Learner Guide and the Lesson Plan are in [`courseware/`](courseware/).

## Assessment

- **Written Assessment (WA)** — 16 open-ended short-answer knowledge questions, 1 hour, open book.
- **Case Study (CS)** — one continuous scenario (Nimbus Wellness Pte Ltd) with 4 tasks mapped to LU1–LU4, 1 hour, open book.
- Grading: Competent / Not Yet Competent. Minimum 75% attendance required for funding eligibility.
